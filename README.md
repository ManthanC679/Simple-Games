#  Simple Python Game (Menu Driven)

A beginner-friendly Python project that includes two small games:

* Stone Paper Scissors
* Dice Roll Game
* This project demonstrates basic programming concepts like functions, modules, and random number generation using a simple menu-driven approach.

##  Project Structure
├── #---Main---.py          # Controls the menu and program flow
├── sps.py                  # Contains Stone Paper Scissors logic
├── Dice.py                 # Contains Dice Roll game logic
└── README

# 🔹 main.py

* Acts as the **entry point** of the program
* Displays the menu to the user
* Calls functions from other modules

### 🔹 sps_game.py

* Contains logic for **Stone Paper Scissors**
* Handles user input and game result

### 🔹 dice_game.py

* Contains logic for **Dice Roll Game**
* Generates random numbers and compares results

## Features

* Menu-driven program
* Two interactive games
* Use of functions for clean code
* Random number generation using `random` module

# Concepts Used

* Functions
* Conditional statements (`if-elif-else`)
* Loops (`while`)
* Modules and imports
* Random module
